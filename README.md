Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

Project Workflow
1. Data Cleaning
2. Missing Value Handling

Key Insights
- More passengers died than survived.
- Female passengers had a much higher survival rate.
- First-class passengers had the highest survival rate.
- Most passengers traveled in third class.
